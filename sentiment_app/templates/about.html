ABOUT

RRLIZER?

RRLIZER adalah aplikasi web untuk analisis sentimen review yang dikembangkan sebagai bagian dari skripsi. Aplikasi ini menggabungkan teknik natural language processing (NLP) dengan machine learning untuk memberikan insight mendalam tentang sentimen pelanggan terhadap produk kosmetik.

Teknologi yang Dimanfaatkan

  • Backend: Django (Python Web Framework)
  • Frontend: HTML, CSS, JavaScript
  • Database: SQLite
  • NLP Libraries: NLTK, Sastrawi, IndoCollex
  • Machine Learning: Scikit-learn, NumPy, Pandas

Fitur Utama

  • PRA-PEMROSESAN DATA MENTAH / PRE-PROCESSING
  • SENTIMENT ANALYSIS PADA REVIEW

Dataset

Aplikasi ini dirancang untuk bekerja dengan dataset yang mengandung:

  • Review Tekstual: Review produk kosmetik dalam Bahasa Indonesia
  • Rating Numerik: Skala rating dari 1-5 bintang
