BANTUAN

Panduan Penggunaan

1. Persiapan Data

Sebelum memulai, pastikan data Anda memiliki format yang tepat:

  • Format file: CSV atau XLSX
  • Kolom: review (teks review) dan rating (angka 1-5)
  • Tidak ada header yang kosong
  • Jumlah data minimal: 100 review

2. Pre-Processing Data

Langkah-langkah yang dilakukan sistem:

  1. Upload Dataset: Pilih file CSV/XLSX dari komputer Anda
  2. Pembersihan Teks: Menghapus URL, mention, hashtag, dan karakter spesial
  3. Case Folding: Mengubah semua teks menjadi lowercase
  4. Tokenisasi: Memecah teks menjadi token individual
  5. Normalisasi Slang: Mengubah slang Indonesia menjadi kata baku menggunakan kamus IndoCollex
  6. Stemming: Mengurangi kata ke bentuk dasarnya menggunakan Sastrawi
  7. Stopword Removal: Menghapus kata-kata umum yang kurang informatif
  8. Download Hasil: Data bersih dapat diunduh untuk digunakan di tahap berikutnya

3. Sentiment Analysis

Proses analisis sentimen melibatkan langkah-langkah berikut:

  1. Upload Data Testing: Data uji untuk evaluasi model
  2. Feature Extraction: Mengubah teks menjadi feature vector
  3. Prediksi: Melakukan prediksi pada data
  4. Evaluasi: Menampilkan metrik performa model (Accuracy, Precision, Recall, F1-Score)

Troubleshooting

File tidak bisa diupload

Pastikan format file adalah CSV atau XLSX, ukuran tidak terlalu besar (< 50MB).

Error pada pre-processing

Kemungkinan ada kolom yang kosong atau nama kolom berbeda. Pastikan kolom review dan rating sudah benar.
